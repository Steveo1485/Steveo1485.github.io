Fridge Friend

Final project at Dev Bootcamp, created in a week long sprint with 3 teammates: Annie Sing, Carter Sowers, and Cricket Wallace. Fridge Friend allows you to create collabrative refrigerators, grocery lists and recipes with spouses, roommates, or even friends coming over for a party. Fridge Friend is built on Rails and incoporates Twilio and Yummly APIS, while using jQuery Mobile for the front-end.

Christmas Name Draw

Created to handle my family's annual holiday gift name draw process, Christmas Name Draw allows users to create and update their holiday wishlist, while also showing the wishlist of their randomly selected gift recipeint. Christmas Name Draw is built on Rails and utilizies Rake tasks for random wishlist pairing as well as ActionMailer to send reminder emails leading up to Christmas.
