Fridge Friend

Final project at Dev Bootcamp, created in a week long sprint with 3 teammates: Annie Sing, Carter Sowers, and Cricket Wallace. Fridge Friend allows you to create collaborative refrigerators, grocery lists and recipes with spouses, roommates, or even friends coming over for a party. Fridge Friend is built on Rails and incorporates Twilio and Yummly API, while using jQuery Mobile for the front-end.

Do Something

Do Something is your own personal Yelp. Rate your activities, then when you need an idea of something to do, have Do Something randomly suggest an activity. Do Something runs on Sinatra and uses the Facebook API with OmniAuth for user auth. Created at Dev Bootcamp with Annie Sing, Ryan Hedges, and Robert Taraya.

Christmas Name Draw

Created to handle my family's annual holiday gift name draw process, Christmas Name Draw allows users to create and update their holiday wish list, while also showing the wish list of their randomly selected gift recipient. Christmas Name Draw is built on Rails and utilizes Rake tasks for random wish list pairing as well as ActionMailer to send reminder emails leading up to Christmas.
